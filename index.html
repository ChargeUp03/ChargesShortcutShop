Places To Use It

Clicking on the Places to use it text on a shortcut page takes you here.

Home Screen

Run a Shortcut from your Home Screen by selecting the arrow next to the name, then tapping Add to Home Screen, then Add. Tap the icon to run the Shortcut.

Lock Screen

On recent iOS and iPadOS versions, you can set a Shortcut as one of two buttons on the Lock Screen. Hold the lock screen, tap Customize, tap the '-' button to remove the current action if it's there, tap the '+', search for Run Shortcut, tap Choose, select the Shortcut, then tap Done.

Control Center

On recent versions of iOS and iPadOS, you can add Shortcuts to the Control Center. Hold a blank space or tap the '+' at the top left corner, tap Add a Control, search for Run Shortcut, choose a Shortcut, resize it, then tap anywhere.

Shortcuts App

Run a shortcut from the Shortcuts app by pressing the play icon on the top or bottom of the screen, depending on your OS.
