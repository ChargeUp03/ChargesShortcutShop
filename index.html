Places To Use It

Places to use these Shortcuts

Home Screen

Run a Shortcut from your Home Screen by selecting the arrow next to the name, then tapping Add to Home Screen, then Add. Tap the icon to run the Shortcut.

Lock Screen

On recent iOS and iPadOS versions, you can set a Shortcut as one of two buttons on the Lock Screen.

Control Center

On recent versions of iOS and iPadOS, you can add Shortcuts to the Control Center.

Shortcuts App

Run a shortcut from the Shortcuts app by pressing the play icon on the top or bottom of the screen.
